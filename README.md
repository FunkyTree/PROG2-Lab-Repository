PROG2 Lab Repository
====================

Dieses Repository wird verwendet um die PROG2 Praktikumsaufgaben in PROG2 auszuteilen und abzugeben.

Jedes Praktikum wird in einem eigenen Unterverzeichnis angelegt in welchem die Anleitung und benötigten Dateien enthalten sind:

- [Lab 1 - Tooling](./Lab01-Tooling/README.md)
- Lab 2 - Testing
- Lab 3 - Concurrency 1
- Lab 4 - Concurrency 2
- Lab 5 - I/0 1 JDBC-File
- Lab 6 - I/O 2 Networking

Wie im ersten Praktikum Tooling gelernt, arbeiten Sie mit einem persönlchen `fork` (*yourfork*) dieses Repositories und erledigen und die Aufgaben in ihrem lokalen `clone`.

Um ihre Lösung abzugeben senden Sie einen Pull-Request zu ihrem persönlichen `branch` (*yourlogin*) in diesem Repository (z.B. Pull Request von *yourfork*/master to PROG2-lab-IT.../*yourlogin*).

Als Abgabezeitpunkt gilt der letzte Zeitpunkt des letzten commits in Ihrm pull-request.

Führen Sie anschliesend die Lösung dem Laborassistenten vor um ihre Punkte zu erhalten.
